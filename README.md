# EduPortal

EduPortal is an application designed to help students, instructors, and administrators manage various aspects of academic life including registration, homework assignments, projects, and more.

## Features

- **Student Portal**: Track homework assignments, project deadlines, and course registrations.
- **Instructor Portal**: Manage course materials, assignments, grades, and communicate with students.
- **Administrator Dashboard**: Monitor overall system usage, manage user accounts, and configure system settings.

## Installation

1. Clone the repository from GitHub:

```bash
git clone
```

2. Install dependencies:

```bash
npm install
```


3. Configure environment variables:

Create a `.env` file in the root directory and add the necessary configuration variables:

```bash
PORT=3000
DATABASE_URL=mongodb://localhost:27017/eduportal
SECRET_KEY=your_secret_key
```


4. Start the application:

```bash
npm start
```

5. Open your browser and navigate to `http://localhost:3000` to use EduPortal.

## Usage

- **Student**: Log in to view courses, assignments, and deadlines. Submit homework and track progress.
- **Instructor**: Log in to manage courses, create assignments, grade submissions, and communicate with students.
- **Administrator**: Log in to manage user accounts, configure system settings, and monitor usage statistics.

## Technologies Used

- Node.js
- Express.js
- MongoDB
